$\mathbb{H}$ypnogogic #s

The class of $\mathbb{H}$ypnogogic numbers (simply $\mathbb{H}$ alternatively $\mathbb{N}og$) treats the class of $\mathbb{O}ne$ as a solitary unit $\mathbb{ONE}$ - which represents the first of the hypnogogic numbers:

$$\mathbb{ONE} = \{\mathbb{ONE}|\frac{1}{\mathbb{ONE}}\}$$ $$-\mathbb{ONE} = \{-\frac{1}{\mathbb{ONE}}|-\mathbb{ONE}\}$$ $$\mathbb{ZERO}=\{-\mathbb{ONE}|\mathbb{ONE}\}$$ $$\mathbb{STAR}=\{\mathbb{ZERO}|\mathbb{ZERO}\}$$

We could then define the class of hypnogogic numbers:

$$\mathbb{H}\lor\mathbb{N}og=\{\mathbb{HUGE}|\mathbb{TINY}\}$$
